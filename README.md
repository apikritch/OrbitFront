### OrbitFront

System Requirements:

- Node.js 20.9.0

Prerequisite:

- npm 10.1.0

Please install pnpm before running the project:

```bash
npm i -g pnpm
```

Then run the following command:

```bash
pnpm install

pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
